# Price Calculator 💰

A simple JavaScript project that calculates the **final price of products** after applying **tax and discount**.

This project demonstrates how to build reusable functions and perform basic arithmetic operations in JavaScript.

---

## 📌 Features

* Calculate the **final product price**
* Add **tax** to the original price
* Apply **discount**
* Reuse a function for multiple products
* Display results using **console output**

---

## 🧠 Concepts Used

This project practices fundamental JavaScript concepts:

* JavaScript Functions
* Parameters
* Return Values
* Variables
* Basic Arithmetic Operations
* Console Output

---
